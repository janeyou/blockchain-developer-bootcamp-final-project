# Marathon Training Planner Dapp

## About

A simple workout plan generator for marathon training

## Get Started

### Run locally

- `npm install`
- `ganache-cli`
- `truffle test`
- `truffle migrate --network development`

### Run frontend

- `yarn start`
- Visit `http://localhost:3000/`
